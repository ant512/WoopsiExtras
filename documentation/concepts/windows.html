Basic Concepts - Windows

Windows are a method of dividing up the display into smaller, self-contained rectangular chunks. The majority of operating systems now include a windowing environment of some sort, so explaining the concept in detail seems rather futile.

Woopsi's windows are capable of the usual set of functions common to all windowing enviroments. They can be opened, closed, moved, hidden, lowered and raised. They can contain other gadgets of various sorts, such as buttons or bitmaps.

Windows are designed to run concurrently. You can open two or more windows at once, and both windows will function independently of each other.
