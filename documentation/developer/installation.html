Developer Guide - Installation

Woopsi does not need to be installed, since using it in a project simply involves copying the source files into the project - see Creating a Woopsi Project for more information on this.

To "install" Woopsi, simply download the archive from the SourceForge project page and unzip it to your hard disk.
